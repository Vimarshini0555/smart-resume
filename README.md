# Smart Resume – AI-Powered Resume Analyzer

## Overview

Smart Resume is an AI-powered web application that helps users analyze and improve their resumes. The system evaluates resume content, identifies missing skills, and provides suggestions to make resumes more professional and ATS (Applicant Tracking System) friendly.

## Features

* Upload and analyze resumes.
* Extract skills and important information automatically.
* AI-based resume evaluation and feedback.
* ATS-friendly recommendations.
* User-friendly web interface.
* Fast and efficient resume processing.

## Technologies Used

* Python
* Flask
* HTML, CSS, JavaScript
* AI/NLP Techniques
* Docker

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Vimarshini0555/smart-resume.git
```

2. Navigate to the project directory:

```bash
cd smart-resume
```

3. Create a virtual environment:

```bash
python -m venv venv
```

4. Activate the virtual environment:

```bash
venv\Scripts\activate
```

5. Install dependencies:

```bash
pip install -r requirements.txt
```

6. Run the application:

```bash
python app.py
```
